# Task 3 — Algorithms and Data Structures

Решения алгоритмических задач с использованием бинарного поиска, жадных алгоритмов и хеш-структур.  
Включает реализацию на Python, тесты (pytest) и пояснения <3

---

## Задачи

### 1. Найти корень числа (ближайшее целое)

Необходимо написать функцию, которая находит целую часть квадратного корня числа (округление вниз).

**Что нужно сделать:**
- Реализовать решение с использованием бинарного поиска
- Не использовать встроенные функции извлечения корня
- Объяснить сложность O(log n) и выбор границ поиска

---

### 2. Два ксерокса

Как быстро можно сделать N копий документа, если есть два ксерокса с разной скоростью (x и y минут за копию)?

**Что нужно сделать:**
- Реализовать бинарный поиск по времени
- Объяснить проверку достаточности времени

---

### 3. Накормить животных

В зоопарке есть животные, каждое требует определённое количество еды.  
Каждая порция может накормить только одно животное.

**Что нужно сделать:**
- Реализовать жадный алгоритм (через сортировку)
- Объяснить, почему стратегия даёт максимум

---

### 4. Найти лишний символ

Даны строки `a` и `b`, где `b` получена из `a` с добавлением одной буквы.

**Что нужно сделать:**
- Решить за O(n)
- Использовать подсчёт символов или XOR

---

### 5. Two Sum (сумма двух элементов массива)

Дан массив и число `target`. Нужно найти два элемента, сумма которых равна `target`.

**Что нужно сделать:**
- Реализовать решение за O(n)
- Использовать хеш-таблицу
- Объяснить, как избегается двойной цикл
