Сначала мы считаем, что первая копия уже сделана за min(x, y) минут, поэтому остаётся сделать n - 1 копий. Далее для 
некоторого времени t считаем, сколько копий успеют сделать оба ксерокса: первый -- t // x, второй -- t // y. Суммарно это 
t // x + t // y. Если это количество больше или равно n - 1, то времени достаточно, иначе недостаточно.
